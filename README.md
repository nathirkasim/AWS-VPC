**Scalable AWS Web Application (Terraform)**
This project demonstrates a production-grade, scalable web application architecture on AWS built using Infrastructure as Code (IaC) with Terraform.

**Key Features**
Multi-tier VPC architecture (public & private subnets)
Application Load Balancer (ALB) for traffic distribution
Auto Scaling Group (ASG) for dynamic scaling
Amazon RDS (MySQL) for managed database
Amazon S3 for storage
CloudWatch + SNS for monitoring & alerts
Secure design with IAM roles & security groups
**Tech Stack**
AWS (EC2, ALB, RDS, S3, CloudWatch, SNS)
Terraform (~5.0)
Ubuntu + Apache2
**deployment**
terraform init
terraform plan
terraform apply
**highlights**
High availability across multiple AZs
Fully automated provisioning (IaC)
Scalable and secure cloud-native design
